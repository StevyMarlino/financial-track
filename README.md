# Description

Application de génération de reférencement naturel

## Environnement de développement

### Pré-requis
 
 * PHP 7.4
 * Composer
 * PSR-2
 
#### lancer l'environnement de développement
      
          composer install
          npm install
          npm run build
          php artisan serve

#### Lancer les Tests

```bash  
        php artisan test
  ```
